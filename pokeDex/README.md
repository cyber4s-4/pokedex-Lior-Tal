## PokéDex Application
Generates a UI based on data fetched from PokéServer</br>
DO NOT run this application if the server isn't up and running</br>
You'll just be disappointed

### How do I do it?

* Make sure PokéServer is UP
* Open terminal on ~/pokeDex
* `npm ci`
* `npm start`
* That's it.
